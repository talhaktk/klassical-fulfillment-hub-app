# Warehouse Images

Place these image files in this directory to show them in the Services section:

- `warehouse-storage.jpg` → Storage & Receiving service card
- `warehouse-prep.jpg`    → Pick & Pack Fulfillment service card  
- `warehouse-dispatch.jpg` → B2B & Bulk Dispatch service card

Images will auto-display if present, and gracefully hide if missing.
Recommended dimensions: 1200×600px or wider.
